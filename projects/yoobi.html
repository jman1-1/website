YOOBI

2025
react native expo typescript supabase

A task and habit management mobile app built with React Native, Expo, and TypeScript, targeting both iOS and Android. Navigation is file-based via Expo Router, the backend runs on Supabase (PostgreSQL, auth, and cloud storage), and the UI is styled with NativeWind (Tailwind CSS ported to React Native). Animations use react-native-reanimated and push notifications are handled through expo-notifications.

Users complete tasks from three sources: custom tasks they create, admin-curated presets, and community-submitted tasks shared by other users. Each task is broken into steps that can include timers, images, and per-step push notification triggers. Alongside tasks, the app ships MedTrak, a medication tracking module where users log dosage frequency, time of day, and food requirements, with recurring device notifications as reminders.

What sets YOOBI apart is its progression and cosmetics layer. Completing tasks and hitting milestones awards achievements and earns tokens, spendable in a built-in marketplace for themes, app icons, and other cosmetics. Themes are fetched from Supabase and applied globally at runtime, giving each user a distinct look. The app also features a leaderboard, a community feed for browsing and reporting shared tasks, and an admin panel for moderation. A recurring mascot called "Yoobi Man" appears throughout the UI with multiple expressive sprite states.
